# 文件结构
```
---Assets
  |---Effects
  |---Music(bgm)
  |---Sounds(音效)
  |---Textures(贴图)
---Common
  |---Command(指令)
  |---Configs(配置)
  |---GlobalBuffs(全局buff)
  |---GlobalNPCs(全局NPC)
  |---GlobalProjectiles(全局弹幕)
  |---GlobalTiles(全局物块)
  |---GlobalWalls(全局墙)
  |---ItemDropRules(物品掉落规则)
  |---Players(玩家)
  |---Systems(系统)
  |---UI(UI)
---Content
  |---Biomes(群系)
  |---Buffs(Buff)
  |---DamageClasses(伤害类型)
  |---Dusts(粒子)
  |---EmoteBubbles(表情泡泡)
  |---Hairs(发型)
  |---Items(物品)
     |---Accessories(饰品)
     |---Ammo(弹药)
     |---Armor(装备)
     |---Consumables(消耗品)
     |---Mounts(坐骑)
     |---Placeable(放置物，在手中)
     |---Tools(工具)
     |---Weapons(武器)
     |---Materials(材料)
  |---Mounts(坐骑、矿车)
  |---NPCs(NPC)
     |---Gores(碎片贴图)
  |---Pets(宠物)
  |---Prefixes(前缀)
  |---Projectiles(弹幕)
  |---TileEntities(物块实体)
  |---Tiles(物块，在地面)
     |---Ores(矿物)
  |---Walls(墙)
```
